#### NodeJs基础介绍及配置

#### Nodejs介绍

简单说，Node.js就是运行在服务端的JavaScript。

- Node.js是基于Chrome JavaScript运行时建立的一个平台。
- Node.js是一个事件驱动I/O服务端JavaScript环境，基于Google的V8引擎，V8引擎执行JavaScript的速度很快，性能很好。

#### 环境安装

##### 1.Node.js

```
brew install node@8
```

##### 2.安装npm

```
npm install npm@5.6.0 -g
```

##### 3.检查Node和npm是否已经安装

```
node -v 
npm -v
```

##### 升级Node操作

```
npm install -g  n
```
